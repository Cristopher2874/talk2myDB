# talk2myDB
Contains two demos for natural language to SQL and NL to graph DB

Sql scripts provide first hand testing of the actual DB queries.

Using `PGQL 2.1 Specification` Important to skills generation

[Specification](https://pgql-lang.org/spec/2.1/#assigning-variable-name-to-select-expression)
